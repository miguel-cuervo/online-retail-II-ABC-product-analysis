# Online Retail II - ABC Product Analysis
Este repositorio documenta el proceso de limpieza, transformación y análisis exploratorio del conjunto de datos "online_retail_II", extraído de Kaggle, utilizando principalmente Power Query en Excel.
Este proyecto sigue el marco de trabajo de análisis de datos del Certificado Profesional de Análisis de Datos de Google, con los pasos Ask, Prepare, Process, Analyze, Share and Act.
## Ask (Preguntar)
El dataset original cuenta con una gran cantidad de transacciones de una tienda en línea basada en el Reino Unido. Sin embargo, los datos "crudos" presentan inconsistencias que impiden un análisis fiable, desde pruebas hasta errores y devoluciones.
